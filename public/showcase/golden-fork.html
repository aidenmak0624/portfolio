Why These Choices?

Next.js 16 App Router

Server-side rendering for SEO and fast initial loads. API routes co-located with frontend â€” single deployment. React Server Components reduce client bundle size.

RAG over Fine-Tuning

Menu data changes frequently. RAG with Pinecone allows real-time menu updates without retraining a model. Embeddings capture semantic meaning â€” "something light" matches salads and soups.

In-Memory Store

MVP-appropriate choice for rapid iteration. Orders, sessions, and feedback stored in JavaScript Maps. Easily swappable to PostgreSQL/Redis for production scale without changing the API layer.
